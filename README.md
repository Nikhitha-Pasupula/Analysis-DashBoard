Food Waste & Inventory Analysis

### Overview
Analyzed a Kaggle food dataset to identify waste patterns, 
inventory inefficiencies, and consumption trends using 
Excel and Power BI.

### Objectives
- Identify high-waste food categories
- Visualize inventory and consumption trends
- Provide actionable recommendations to reduce food waste

### Tools Used
- Excel (Data Cleaning, Power Query, Pivot Tables)
- Power BI (Interactive Dashboard, Data Modelling)

### Dataset
- Source: Kaggle
- https://www.kaggle.com/ [add your exact dataset link]

### Key Findings
- Identified top 3 high-waste food categories
- Built interactive dashboard tracking waste vs inventory levels
- Uncovered patterns in food loss across categories
